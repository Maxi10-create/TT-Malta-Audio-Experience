# GitHub Pages veroeffentlichen

1. Auf GitHub ein neues Repository erstellen.
2. Repository beispielsweise `malta-audio-experience-tt` nennen.
3. Alle Dateien aus diesem Paket hochladen und committen.
4. **Settings** oeffnen.
5. Links **Pages** waehlen.
6. Quelle: **Deploy from a branch**.
7. Branch: `main`.
8. Ordner: `/ (root)`.
9. Speichern und einige Minuten warten.

Die Adresse sieht anschliessend etwa so aus:

```text
https://DEIN-BENUTZERNAME.github.io/malta-audio-experience-tt/
```

Nach jeder Aenderung erneut committen. GitHub Pages aktualisiert die Webseite normalerweise innerhalb weniger Minuten.
